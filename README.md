# ClawCross

ClawCross is a local-first multi-agent orchestration project.

This initial version only establishes the repository, configuration templates,
and the basic project contract for later runtime, frontend, workflow, and
assistant capabilities.

## Scope In This Version

- Repository metadata and licensing
- Environment and dependency templates
- Minimal operator-facing documentation
- Empty runtime data roots

## Planned Build-Out

Later versions will add:

- local service bootstrap and configuration scripts
- backend agent APIs and web UI
- group chat and mobile surfaces
- OASIS workflows and OpenClaw integration
- team creator, presets, memory, and self-evolution

## Layout

- `config/` configuration templates
- `data/` runtime data root
- `docs/` operator documentation
- `src/` backend services
- `frontend/` web assets
- `selfskill/` local setup helpers
